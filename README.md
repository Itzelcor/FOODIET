# Subsistema de Estadísticas – Frontend

**Desarrollado por:** Itzel Bethania  
**Fecha:** Junio 2026

## Páginas desarrolladas

- `estadisticas-dashboard.html` – Panel de rendimiento con métricas y gráfica
- `estadisticas-generar.html` – Formulario para generar informes
- `estadisticas-informe.html` – Vista de informe detallado por profesional
- `estadisticas-progreso.html` – Panel de progreso individual del paciente

## Tecnologías

- Bootstrap 5.3
- HTML5 / CSS3
- CSS compartido: `css/estilos.css`

## Notas

Las páginas siguen el mismo navbar, footer y clases CSS que el resto del equipo para garantizar cohesión visual.
